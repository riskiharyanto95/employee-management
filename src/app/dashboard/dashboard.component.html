1. Untuk akses menu mohon klik icon menu yang berada di pojok kiri atas.

  2. Untuk logout mohon klik icon person yang berada di pojok kanan atas, kemudian klik icon logout

  3. List Employee, default sorting berdasarkan id terakhir. Jadi ketika sukses create employee, data employee akan berada di paling atas. Hal ini akan berubah apabila ada aktivitas sorting, paging & filtering.

  4. Berdasarkan soal, kolom deskripsi tipe data datetime. Jadi secara form saya bikin datepicker.

  5. Untuk menampilkan datepicker, mohon klik icon today pada setiap field.
